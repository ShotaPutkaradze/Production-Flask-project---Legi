# Проект: Журнал смены (Легкая версия)

## Описание:

Этот веб-интерфейс предназначен для операторов на производстве, чтобы они могли быстро и просто вносить данные о работе смены. Вся информация сохраняется в структуре на основе CSV-файла с номенклатурой.

## Функции:

* Выбор завода (Тбилиси / Кобулети)
* Выбор стороны производства (Мокрая / Сухая)
* Последовательный выбор:
  → группы продукции
  → подгруппы
  → категории
  → наименования (номенклатуры)
* Возможность указать количество и ФИО оператора
* Альтернативный путь "Не нашел номенклатуру" — позволяет ввести данные вручную

## Технологии:

* Python 3.10+
* Flask
* Pandas
* HTML (Jinja2 шаблоны)

## Файлы:

* app.py             — основной сервер приложения
* nomenclature.csv   — справочник всей продукции
* templates/
  ├── index.html               — выбор завода
  ├── side.html                — выбор стороны
  ├── subgroup.html            — выбор подгруппы
  ├── category.html            — выбор категории
  ├── product.html             — выбор наименования
  ├── manual\_input.html        — ручной ввод
  └── submit.html              — подтверждение ввода

## Установка и запуск:

1. Установить зависимости:
   pip install flask pandas

2. Запустить сервер:
   python app.py

3. Открыть браузер:
   [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Замечания:

* Данные номенклатуры хранятся в `nomenclature.csv`, не забудьте его обновлять.
* В продакшн-среде рекомендуется использовать базу данных (PostgreSQL, SQLite).
* Возможна интеграция с Power BI или PowerApps в будущем.
